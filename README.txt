This plugin allows for the search and embedding of information from the Knowledge Services Open API.

== Description ==

The Knowledge Services Open API provides easy programmatic access to tens of thousands of thematically organised research documents that are freely available online.

This plugin allows for the search and embedding of information from the <a href="http://api.ids.ac.uk/about/">Knowledge Services Open API</a>.


== Installation ==

The plugin includes everything required for it's use.

1. Upload the entire 'wp-eldis` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
